## Overview:
This lab assignment is focused on building a web-based dashboard to manage and interact with computer model data and associated user portfolios. The lab is divided into two parts:

1. **Part 1**: Develop the core interface and user interaction for displaying and managing user and computer stock data.
2. **Part 2**: Implement a detailed computer information viewer using advanced JavaScript techniques.

## Prerequisites:
- Proficiency in HTML, CSS, and JavaScript (ES6) with basic web development concepts.
- Familiarity with JSON data, array functions, classes, and modules.
- Knowledge of Git for version control.

## Objectives:
- Create a dynamic and interactive user interface for managing computer data.
